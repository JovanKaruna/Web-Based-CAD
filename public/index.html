User Manual

  • Change color : Klik dan ubah warna pada color pallete.
  • Make model : Pilih radio button dari model yang ingin dibuat, jangan checklist edit, klik mouse pada canvas untuk membuat model yang dipilih.
  • Edit model : Pilih radio button dari model yang ingin diedit, checklist edit, klik simpul pada sebuah model yang ada pada canvas sesuai dengan pilihan model di radio button untuk melakukan edit (mengubah arah dan ukuran).
  • Clear canvas : Klik button clear canvas untuk menghapus seluruh gambar yang ada pada canvas.
  • Save canvas : Klik button save untuk menyimpan canvas pada sebuah file download.
  • Load canvas : Klik button load untuk menampilkan canvas yang pernah di save.
